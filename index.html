集成学习

发表于 2019-09-29 | 分类于 机器学习

集成学习通过构建并结合多个学习器来完成学习任务。一般的结构：先产生一组“个体学习器”（一般为弱学习器），再使用某种策略将多个个体学习器结合起来，形成一个强学习器。集成学习可以用于分类问题集成，回归问题集成，特征选取集成，异常点检测集成等等。

集成学习有两个主要的问题需要解决，第一是如何得到若干个个体学习器，第二是如何选择一种结合策略，将这些个体学习器集合成一个强学习器。

集成学习算法主要的类别：Bagging，Boosting以及Stacking

Bagging

在Bagging方法中，利用bootstrap sampling（自助采样法，有放回的采样）方法从整体数据集中采取有放回抽样得到 $$N$$ 个数据集，在每个数据集上学习出一个模型，最后的预测结果利用$$N$$个模型的输出得到，具体地：分类问题采用$$N$$个模型预测投票的方式，回归问题采用$$N$$个模型预测平均的方式。

随机森林

随机森林在以决策树为基学习器构建Bagging集成的基础上，进一步在决策树的训练过程中引入了随机属性的选择。使得随机森林中的随机包含两个部分，一个是训练样本的随机选择，另外一个是样本属性的随机选择（可以欧有效的防止过拟合）。

随机森林在预测的时候，每一棵树的都对输入进行预测，最后进行投票，哪个类别多，输入样本就属于哪个类别。这就相当于前面说的，每一个分类器（每一棵树）都比较弱，但组合到一起（投票）就比较强了。

Boosting

Boosting 是一族可将弱学习器提升为强学习器的算法，算法的工作机制：先从初始训练集训练一个基学习器，在根据基学习器的表现对样本分布进行调整，使得先前基学习器做错的训练样本在后续受到更多的关注，然后基于调整后的训练样本的分布来训练下一个基学习器，如此重复，直到基学习器的个数达到指定的个数 $$T$$，最终将这 $$T$$个基学习器进行加权结合

Boosting系列算法里最著名算法主要有AdaBoost算法和提升树(boosting tree)系列算法。提升树系列算法里面应用最广泛的是梯度提升树(Gradient Boosting Tree)

GBDT

Stacking

Stacking 方法是指训练一个模型用于组合其他各个模型。首先我们先训练多个不同的模型，然后把之前训练的各个模型的输出为输入来训练一个模型，以得到一个最终的输出。理论上，Stacking 可以表示上面提到的两种Ensemble 方法，只要我们采用合适的模型组合策略即可。但在实际中，我们通常使用 logistic 回归作为组合策略。

降维

发表于 2019-09-29 | 分类于 机器学习

​ 一般在观测和收集到的数据样本是高维的，但是在学习和处理当中，只需要其中的某些相关性较大的维度的数据信息。所以，一般会对数据进行降维处理，通过某种数学变换将高维属性的数据转化为低维属性空间的数据。通过降维,我们希望减少冗余信息所造成的误差，提高识别（或其他应用）的精度。或者通过降维算法来寻找数据内部的本质结构特征。有些算法没有对数据进行降维处理很难实现数据的准确的分类等操作。

​ 常见的降维方法有：PCA（主成分分析）、LDA（Linear Discriminant Analysis）、LLE（局部线形嵌入）、Laplacian Eigenmaps（拉普拉斯特征映射）

PCA

Principal Component Analysis(PCA)是最常用的线性降维方法，它的目标是通过某种线性投影，将高维的数据映射到低维的空间中表示，并期望在所投影的维度上数据的方差最大，以此使用较少的数据维度，同时保留住较多的原数据点的特性。（PCA是丢失原始数据信息最少的一种线性降维方式）

PCA把原先的n个特征用数目更少的m个特征取代，新特征是旧特征的线性组合，这些线性组合最大化样本方差，尽量使新的m个特征互不相关。从旧特征到新特征的映射捕获数据中的固有变异性。

PCA追求的是在降维之后能够最大化保持数据的内在信息，并通过衡量在投影方向上的数据方差的大小来衡量该方向的重要性

注：特征中心化。即每一维的数据都减去该维的均值。这里的“维”指的就是一个特征（或属性），变换之后每一维的均值都变成了0。（减均值操作）

LDA

Linear Discriminant Analysis (也有叫做Fisher Linear Discriminant)是一种有监督的（supervised）线性降维算法。与PCA保持数据信息不同，LDA是为了使得降维后的数据点尽可能地容易被区分！

类间距大，类内距小

（注：LDA和PCA是线形降维算法，也是一种谱方法）

LLE

Locally linear embedding（LLE）[1] 是一种非线性降维算法，它能够使降维后的数据较好地保持原有流形结构

算法主要包含以下三个步骤：

（1）寻找每个样本点的k个近邻点；

（2）由每个样本点的近邻点计算出该样本点的局部重建权值矩阵；

（3）由该样本点的局部重建权值矩阵和其近邻点计算出该样本点的输出值。

Laplacian Eigenmaps

它的直观思想是希望相互间有关系的点（在图中相连的点）在降维后的空间中尽可能的靠近。

什么是决策树

决策树是一种常见的机器学习算法，既可以用于分类任务，也可以实现回归任务；决策树是基于树结构进行决策的。符合人类解决实际问题的思考逻辑。对于一个问题，会进行一系列的判断，最后得出最终的决策（分类结果或回归量）。
