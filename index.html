过拟合与欠拟合

发表于 2019-10-08 | 分类于 基础知识

过拟合

表现：在训练集上的性能很好，但是在验证集上的性能很差，网络学习到了数据的特例信息，降低了网络的泛化能力，此时模型的方差较大

造成原因：模型太复杂，数据简单却选择了大模型，数据量有限

解决办法：

  1. 增加数据量
  2. 降低网络的复杂度
  3. dropout
  4. early stoping（早停）
  5. 参数正则化，降低过拟合，提高泛化能力（正则化的目的是用来防止过拟合的，所以增大正则化系数可以解决过拟合，减小则可以解决欠拟合）
  6. 使用BN（Batchnorm本质上是解决反向传播过程中的梯度问题）

欠拟合

表现：在训练集以及测试集上同时具有较高的误差，此时模型的偏差较大

造成原因：模型太简单，数据太复杂，网络没有学习到数据内容

解决办法：

  1. 增加数据特征项
  2. 适当增加网络的复杂度
  3. 增加模型学习率
  4. 加载预训练模型，微调网络结构
  5. 使用集成学习方法、或者使用多模型融合

梯度消失与梯度爆炸

发表于 2019-10-08 | 分类于 基础知识

其实梯度爆炸和梯度消失问题都是因为网络太深，网络权值更新不稳定造成的，本质上是因为梯度反向传播中的连乘效应

梯度消失

主要原因：在反向传播的过程中，每层的梯度大于0小于1（sigmoid函数的最大梯度为0.25），经过很深的网络进行反向传播的时候，就会造成梯度接近0的情况，造成梯度消失

梯度爆炸

主要原因：网络太深、初始化权重太大、损失函数设计不合理、权重更新不稳定等原因造成的

解决办法

  1. 降低网络的深度，如果需要深度网络，可以使用resnet中的短链接结构以防止梯度消失（resnet中的短链接可以改善梯度消失问题，但是resnet主要不是解决梯度消失问题的https://www.jianshu.com/p/09643588f373）
  2. 激活函数使用ReLU，代替sigmoid函数
  3. 使用BN（Batchnorm本质上是解决反向传播过程中的梯度问题）
  4. 使用权重正则化、梯度剪切（一般针对梯度爆炸使用）
  5. 加载预训练模型，在预训练模型上进行微调

聚类

发表于 2019-10-07 | 分类于 机器学习

聚类是将样本集合中相似的样本分配到相同的类，不相似的样本分配到不同的类。聚类的时候，样本通常是欧式空间中的向量，类别不是事先给定的，而是从数据中自动发现的，但是类别的数量是给定的。如果样本只能属于某一个类别，则称为硬聚类，如果一个样本可以属于多个类别，则称为软聚类。层次聚类、K均值聚类是硬聚类方法；高斯混合模型EM算法是软聚类算法

聚类的核心概念是相似度或距离，相似度或距离的定义有多种。不同的定义方法直接影响聚类的结果，所以相似度的度量方法的选择是聚类的根本问题

主要的聚类方法有以下三种：原型聚类、密度聚类、层次聚类

相似度计算方法

闵可夫斯基距离

给定样本集合（是维实数向量空间中点的集合），其中，$x_{i}=(x_{1i},x_{2i},…,x_{mi})^T$，$x_{j}=(x_{1j},x_{2j},…,x_{mj})^T$，样本$x_{i}$与样本$x_{j}$的闵可夫斯基距离定义为：

$d_{ij}=(\sum_{k=1}^{m}|x_{ki}-x_{kj}|^{p})^{\frac{1}{p}}$，$p \geq1$

当$p=2$时称为欧式距离；当$p=1$时称为曼哈顿距离；当$p=\infty$时称为切比雪夫距离，即取各个坐标数值差的绝对值的最大值：$d_{ij}=max_{k}|x_{ki}-x_{kj}|$。

闵可夫斯基越大相似度越小，距离越小相似度越大

原型聚类

原型聚类称为“基于原型的聚类”，一般假设聚类结构能够通过一组原型刻画，在现实聚类任务重极为常见。通常情况下，算法先对原型进行初始化，然后对原型进行更新迭代求解，采用不同的原型表示、不同的求解方式，将产生不同的算法。

K 均值算法

K均值聚类是基于样本集合划分的聚类算法。K均值聚类将样本集合划分为K个子集，构成K个类，将n个样本划分到K个类中，每个样本到其所属类的中心距离最短。每个样本只属于某一类，所以，K均值聚类算法属于硬聚类算法。

算法流程：

输入：$n$个样本的集合$X$；

输出：样本集合的聚类$C^{*}$

  1. 初始化，令 t=0，随机选择开个样本点作为初始聚类中心 $m^{(0)}=(m^{(0)}_{1},…,m^{(0)}_{l},…,m^{(0)}_{k})$。
  2. 对样本进行聚类。对固定的类中心$m^{(t)}=(m^{(t)}_{1},…,m^{(t)}_{l},…,m^{(t)}_{k})$，其中$m_{l}^{(t)}$，为类$G_{l}$，的中心，计算么个样本到类中心的距离，将每个样本指派到与其最近的中心的类中，构成聚类结果$C^{(t)}$
  3. 计算新的类中心，对聚类结果$C^{(t)}$，计算当前各个类中的样本的均值，作为新的类中心$m^{(t)}=(m^{(t+1)}_{1},…,m^{(t+1)}_{l},…,m^{(t+1)}_{k})$
  4. 如果迭代收敛或者符合停止条件，输出$C^{*}=C^{t}$。否则，令 t=t+1，返回第2步。

由于K均值使用的是迭代算法求解，所以不能保证得到全局最优解。最后的输出结果是和初始化结果的设置相关，初始化中心的选择直接影响到聚类的结果，在求解过程中会发生偏移，但是一般不会偏移很多。

对于初始化中心的选择：可以使用层次聚类对样本进行聚类，得到K个类是停止，然后从每个类别中选择一个与中心距离最近的点（使用层次聚类的结果作为K均次算法的初始值）

对于K的选择：K均值算法中的K值需要事先指定，然而在实际的应用中最优的K值是不知道的。所以，可以尝试不同的K值聚类，检验各自得到的聚类的质量，推测最优的K值，聚类的结果可以使用类的平均直径来衡量。一般的类别数目变小，平均直径会增加，类别数变大超过某个值以后，平均直径会不变，而这个值则是最优的K值。

学习向量化

学习向量化，简称LVQ，也是试图找到一组原型向量来刻画聚类结构，但是与一般的聚类算法不同的是，LVQ假设数据样本带有类别标记，学习过程利用样本的监督信息来进行非要在辅助聚类。

层次聚类

层次聚类假设类别之间存在层次结构，将样本聚到层次化的类中。层次聚了有聚合或自下而上的聚类、分裂或自上而下聚类两种方法。因为每个样本只属于某一类，所以，层次聚类属于硬聚类算法。

算法流程：

输入：n个样本组成的样本集合以及样本之间的距离

输出：对样本集合的一个层次化聚类

  1. 计算n个样本两两之间的欧式距离$\{d_{ij}\}$，记做矩阵$D=[d_{ij}]_{n\times n}$。
  2. 构造n个类，每个类只包含一个样本。
  3. 合并类间距最小的两个类，其中最短距离为类间距离，构建一个新类。
  4. 计算新类与当前各类之间的距离，若类的个数为1，终止计算，否则会到第3步。

SVM

发表于 2019-10-02 | 分类于 机器学习

SVM（支持向量机）是一种常用的二分类模型，它的基本模型是定义在特征空间上的间隔最大的线性分类器，间隔最大化使其有别于感知机。在使用核函数的时候可以将SVM转化成非线形分类器。SVM的学习算法是求解凸二次规划的最优化算法，等价于正则化的合页损失函数的最小化问题。

支持向量机学习方法包含多种不同的学习模型：

  1. 线形可分支持向量机：训练数据线形可分，可以通过硬间隔最大化学习分类器
  2. 线形支持向量机：训练数据近似线形可分，可以通过软间隔最大化学习分类器
  3. 非线形支持向量机：训练数据线形不可分，可以通过使用核技巧以及软间隔最大化学习分类器

线形可分支持向量机（硬间隔）

分类器学习的目标是在特征空间中寻找一个分离超平面，将样本划分到不同的类，分离超平面将样本划分成两个部分，一部分是正类，一部分是负类，法向量指向的一侧为正类，另外一侧为负类。

一般的训练数据线形可分的时候，存在无穷多个分离超平面可将两类数据分离开，感知机使用误分类最小的策略，求得分离超平面，不过这是的解有无穷多个，线性可分支持向量机，利用间隔最大化求最优分离超平面，解则是唯一的。

函数间隔与几何间隔

对于一个点距离分离超平面的远近可以表示分类预测的确定程度，在超平面确定的情况下，能够相对的表示点距离超平面的远近，而的符号与类比标记的符号是否一致表示分类是否正确。所以可以用量来表示分类的正确定性与确定度，这就是函数间隔

函数间隔可以表示分类的正确定性与确定度，但是选择分离超平面时候只有函数间隔是不够的，因为只要成比例的修改，和，超平面并没有改变，但是函数间隔却成对应倍数的改变，所以我们需要对分离超平面进行限制，如，使得间隔是确定的，这是函数间隔称为了几何间隔。

核方法：通过一个非线性变换将原空间的数据转换到一个新的空间，；然后在一个新的空间使用一个线形分类学习方法从训练数据中学习分类模型。

SVM 算法常见问题总结

  1. SVM对缺失数据的敏感性

    缺失数据是指的数据的特征不完整，向量数据不完整。SVM没有处理缺失值的策略（决策树有）。而SVM希望样本在特征空间中线性可分，所以特征空间的好坏对SVM的性能很重要。缺失特征数据将影响训练结果的好坏。

  2. SVM与LR的区别与联系
    联系：（1）分类（二分类） （2）可加入正则化项
    区别：（1）LR–参数模型；SVM–非参数模型？（2）目标函数：LR—logistical loss；SVM–hinge loss （3）SVM–support vectors；LR–减少较远点的权重 （4）LR–模型简单，好理解，精度低，可能局部最优；SVM–理解、优化复杂，精度高，全局最优，转化为对偶问题—>简化模型和计算 （5）LR可以做的SVM可以做（线性可分），SVM能做的LR不一定能做（线性不可分）

决策树

发表于 2019-10-01 | 分类于 机器学习

决策树是一种常见的机器学习算法，既可以用于分类任务，也可以实现回归任务；决策树是基于树结构进行决策的。符合人类解决实际问题的思考逻辑。对于一个问题，会进行一系列的判断，最后得出最终的决策（分类结果或回归量）。

决策树的优点：模型具有可读性，分类速度快；学习时，利用训练数据根据损失函数最小化的原则简历决策树

决策树模型的生成通常包含三个过程：1、特征选择；2、决策树的生成；3、决策树剪枝

决策树学习本质上是从训练数据集中归纳出一组分类规则，与训练数据集不相矛盾的决策树可能有很多，也可能一个都没有，我们需要的是一个与训练数据矛盾较小的决策树，同时具有较强的泛化能力。决策树的学习是由训练数据估计条件概率模型。决策树学习的损失函数通常是正则化的极大似然函数。决策树学习的算法通常是一个递归的选择最优特征的过程，并根据特征对训练数据集行分割，使得各个子数据集有一个很好的分类的过程，这一过程对应着对特征空间的划分，也对应着决策树的构建过程。决策树构建过程中递归的结束条件为：所有的训练数据子集基本被正确分类；或没有合适的分类特征。

决策树的学习思想主要有：1、ID3；2、C4.5；3、CART

特征选择

特征选择的目的是在构建决策树的时候，选取具有很好分类能力的特征，这样可以有效的提高决策树的学习效率。如果使用一个特征进行分类，分类的结果和随机分类的结果相差不大，则可以认为这个特征不具有分类能力。通常特征选择的标准是信息增益或者信息增益比。

决策树的生成

ID3（信息增益）

ID3算法是在决策树的各个节点上应用信息增益准则选择特征，递归的构建决策树。

从根节点开始，对节点计算所有可能的特征的信息增益，选择信息增益最大的特征作为节点的特征，有该特征的不同取值构建子节点，再对子节点递归的调用以上的方法，构建决策树；直到所有的特征的信息增益均很小或没有特征可以选择为止。最后得到一个决策树。ID3 相当于使用极大似然估计法进行概率模型的选择。

C4.5（信息增益率）

C4.5 算法与 ID3 算法显示，只是在进行特征选择的时候使用的是信息增益比

决策树的剪枝

递归生成的决策树，对与训练数据一般具有很好的划分能力，但是对测试数据可能并没有很好的泛化能力，即发生了过拟合的现象，需要对决策树进行剪枝操作，降低决策树的复杂度，从而提高决策树的泛化能力。决策树的剪枝一般可以分为“预剪枝”和“后剪枝”。预剪枝是指在决策树的生成的过程中，对每个节点在划分前先进行估计，若当前节点的划分不能带来决策树泛化性能的提升，则停止划分，并将当前节点标记为叶节点。 后剪枝则是使用训练集先生成一个完整的决策树，然后自底向上的对非叶节点进行考察，如果将该节点对应的子树替换为叶节点能带来决策树泛化性能的提升，则将该子树替换为叶节点。（所有的泛化性能提升的判断是根据验证集的结果进行判断的）

CART

CART 既可以用于分类，也可以用于回归的任务。CART 是在给定输入随机变量条件下输出随机变量条件概率分布的学习方法，CART假设决策树是二叉树。CART决策树的生成是递归地构建二叉决策树的过程，对于回归树，使用平方误差最小化准则；对于分类树，使用基尼指数最小化准则，进行特征选择，生成二叉树

朴素贝叶斯是基于贝叶斯定理与特征条件独立假设的分类算法。对于给定的训练数据集，首先基于特征条件独立假设学习输入/输出的联合概率分布；然后基于此模型，对于给定的输入，利用贝叶斯定理求出后验概率最大的输出（基于贝叶斯定理使用先验概率求后验概率，将最大的后验概率对应的类别作为样本的类别）。

朴素贝叶斯实际上学习到生成数据的机制，所以属于生成模型，条件独立假设等于是说用于分类的特征在类确定的条件下都是条件独立的。

朴素贝叶斯分类时，对于给定的输入，通过学习到的模型计算后验概率分布，将后验概率最大的类作为的类别输出，后验概率计算根据贝叶斯定理：（注：一般在计算的时候，不考虑分母，因为分母都一样，可以忽略不计算）

后验概率最化的含义：将输入的样本分到后验概率最大的类中，等价于期望风险最小。
